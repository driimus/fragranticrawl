# fragranticrawl
Python webcrawler for fragrantica

## Pre-requisites
- Python 3.7+
- Scrapy

## Usage
`scrapy runspider [spider_name.py]`

## What is being collected
The following information about a fragrance is selected:
- name
- brand
- perfumers
- notes
- gender
- year of releasee
- ~~collection~~
